# Support

## How to Get Help

- **Bug Reports & Feature Requests**: Use the GitHub issue templates (see [.github/ISSUE_TEMPLATE/](.github/ISSUE_TEMPLATE/)).
- **Discussions & Questions**: GitHub Discussions (enable in repo settings if needed) or start a thread in relevant issues.
- **Security Issues**: Follow [SECURITY.md](SECURITY.md) — do not use public issues.
- **Documentation**: Start with [README.md](README.md) and [docs/ARCHITECTURE.md](docs/ARCHITECTURE.md).

## Community

E-COS is part of the larger [Codessa](https://github.com/AvaPrime/codessa) ecosystem focused on building trustworthy, event-sourced cognitive infrastructure.

We are a small, principle-driven team. Responses may take a few days. Thoughtful, well-researched questions with context receive priority.

## Commercial Support

For enterprise use, custom integrations, or priority support, contact ava.prime@outlook.com.
